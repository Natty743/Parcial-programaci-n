# Parcial-programacion
Estructura:
- frontend/
  - index.html
  - styles.css
  - script.js
- backend/
  - server.js
- package.json
- README.md

Instrucciones para ejecutar:
1. Asegúrate de tener Node.js instalado (v14+ recomendado).
2. En la raíz del proyecto, instala dependencias:
   npm install
3. Ejecuta el servidor:
   npm start
4. Abre en el navegador: http://localhost:3000

El servidor sirve los archivos estáticos del frontend y expone un endpoint JSON en /api/message.
